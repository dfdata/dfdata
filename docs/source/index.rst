dfdata文档
==================================
  
dfdata用于下载金融数据，默认使用sqlite数据库保存数据。
将常用的数据保存到本地的优点：

* 不用重复从服务器获取数据
* 减轻服务器的压力
* 稳定性增加
* 不受限制使用数据
